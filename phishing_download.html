Friendly Usage Tip

After downloading the EICAR Test Script (ZIP), your macOS system may automatically extract it, or it may remain as a ZIP file in your Downloads folder. Follow the appropriate instructions below:

If the ZIP file is automatically extracted:

  1. Locate the eicar_test_script.sh file in your Downloads folder.
  2. Open a Terminal window and run the following command:
        
chmod +x ~/Downloads/eicar_test_script.sh && ~/Downloads/eicar_test_script.sh

If the ZIP file is not automatically extracted:

  1. Locate the downloaded eicar_test_script.zip file in your Downloads folder.
  2. Double-click the ZIP file to extract its contents.
  3. Open a Terminal window and run the following commands:
        
unzip ~/Downloads/eicar_test_script.zip -d ~/Downloads/
chmod +x ~/Downloads/eicar_test_script.sh
~/Downloads/eicar_test_script.sh

This process will simulate a phishing attack by running the EICAR test file, which should be detected by your antivirus software as a safe test alert.
